## Simple Portfolio Project

This is a minimal example of a project with just the essential frontmatter fields.

### Tech Stack

- Next.js 14 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- MDX for content management

### Features

- Responsive design
- Dark mode support
- Project filtering
- Fast loading times

The website serves as a platform to showcase design work and attract potential clients.

### Outcome

The new portfolio helped increase client inquiries by 150% and showcased technical capabilities effectively.
